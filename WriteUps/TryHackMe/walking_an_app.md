# Walking an Application
> [TryHackMe Link](https://tryhackme.com/room/walkinganapplication)

When reviewing a website or web application is to discover features that could potentially be vulnerable and attempt to exploit them to assess whether or not they are.
- exploring the website and noting down the individual pages/areas/features with a summary for each one.

## View Page Source
The human-readable code returned to our browser/client from the web server each time we make a request. Made up of: 
- HTML (HyperText Markup Language)
- CSS (Cascading Style Sheets) 
- JavaScript
- Visit `view-source:https://www.google.com/`
